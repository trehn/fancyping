# Colorful ICMP pings for your terminal

```
usage: fancyping [-h] [-i INTERVAL] [-t TIMEOUT] [-D] [-U] [--version] TARGET

Colorful ICMP pings for your terminal

positional arguments:
  TARGET

options:
  -h, --help            show this help message and exit
  -i INTERVAL, --interval INTERVAL
                        number of seconds between each ping (defaults to 1)
  -t TIMEOUT, --timeout TIMEOUT
                        number of seconds before a ping is considered lost (defaults to 2)
  -D, --no-down-anim    disable animation while TARGET is down
  -U, --no-up-anim      disable animation while TARGET is up
  --version             show program's version number and exit

HOTKEYS

 +/-  change stats interval
  Q   quit
  R   reset stats
```
